# Spring-Boot-Ajax-Jquery-Simple-Example
<li>use jQuery.ajax to send a HTML form request to a Spring REST API and return a JSON response. </li>

## Tools used :

<li>Spring Boot 1.5.1.RELEASE</li>
<li>Spring 4.3.6.RELEASE</li>
<li>Maven 3</li>
<li>jQuery</li>
<li>Bootstrap 3</li>
